# CVA Auctions Scraper

Scrapes **all trucks** from every CVA auction page and outputs structured JSON.

### Fields
- Title
- Registration Number
- Make
- Model
- Body Type
- Manufactured
- Mileage
- Location
- Vendor
- Description
- Image URL
- Details URL

### Run Locally
```bash
npm install
npm start
```

### Deploy to Apify
Push this repo to GitHub → link it in Apify → run as an Actor.
